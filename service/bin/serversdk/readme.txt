该目录下的可执行程序均为业务服务器程序（调用AnyChat Server SDK），需与核心服务器配合才能正常工作。

1、anychatserverdemo		调用AnyChat Server SDK的演示程序，源代码在src\server\c++目录中

2、anychatbusinessserver.jar	调用AnyChat Server SDK的演示程序，有GUI界面，源代码在src\server\java目录中

3、anychatsampleserver.jar	调用AnyChat Server SDK的演示程序，没有GUI界面，适合在控制台运行，源代码在src\server\java目录中
